## Common Questions

**Q: Does it save my choice?**
A: Currently it's session-based. You can add localStorage to persist selection.

**Q: Can I have different themes on different pages?**
A: Yes! You can conditionally set themes per page if needed.

**Q: Will it affect my site performance?**
A: No, it's very lightweight. Just CSS variable changes.

**Q: Can I remove the theme switcher?**
A: Yes! Just comment out the `<ThemeSelector />` component.

**Q: Can customers choose their preferred theme?**
A: Yes! The switcher lets any visitor choose their preferred view.
